📸 PHOTO PLACEMENT GUIDE — AluminoPro
=======================================

Place your photos IN THIS FOLDER: static/uploads/

Use EXACTLY these filenames (case-sensitive):

┌─────────────────────────────────────────────────────────────┐
│  FILE NAME          WHERE IT SHOWS              SIZE         │
├─────────────────────────────────────────────────────────────┤
│  hero.jpg           Homepage big banner          1920x1080   │
│  about.jpg          About us section             800x600     │
│  logo.png           Logo in navbar               200x60      │
│  workshop.jpg       Workshop/factory photo       800x600     │
│  project1.jpg       Project gallery #1           800x600     │
│  project2.jpg       Project gallery #2           800x600     │
│  project3.jpg       Project gallery #3           800x600     │
│  project4.jpg       Project gallery #4           800x600     │
│  project5.jpg       Project gallery #5           800x600     │
│  project6.jpg       Project gallery #6           800x600     │
└─────────────────────────────────────────────────────────────┘

SUPPORTED FORMATS: .jpg  .jpeg  .png  .webp

TIPS:
- Compress photos before uploading (use squoosh.app - free)
- hero.jpg should be your best/most impressive project photo
- project1.jpg ... project6.jpg should be different project types
  e.g. windows, doors, ACP cladding, glass partition etc.
- If a photo is missing, a placeholder will show instead

EXAMPLE — what to name your photos:
  Your aluminum window project photo → rename it to "project1.jpg"
  Your ACP cladding project photo   → rename it to "project2.jpg"
  Your workshop photo                → rename it to "workshop.jpg"
